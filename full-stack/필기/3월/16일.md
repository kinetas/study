# 3월 16일 필기 — screen-implement (화면 구현)

## 핵심요약

HTML 전체 태그(리스트, 테이블, a·img, form, video 등)와 CSS 전 범위(BASIC~MQ)의 예제 파일 인덱스를 정리하였다. JavaScript는 onclick, href="javascript:..." 같은 이벤트 속성으로 실행하거나 script 태그 내부에서 실행하며, DOM을 통해 HTML 요소에 접근하고 조작한다. 변수는 let(재할당 가능)과 const(상수)를 사용하고, 타입은 number·string·boolean·object·null·undefined가 있다. 배열의 forEach·filter·map·reduce와 객체의 key-value 구조 및 동적 DOM 생성(createElement, appendChild)까지 학습하였다.



- HTML 기본 구조와 단축키·확장 플러그인: 문서의 시작 형태와 편집 속도를 높이는 환경 정리.
- block / inline, p, div, span: 태그 배치 방식과 기본 문서 태그 역할 정리.
- ul, ol, li 리스트: 순서 없는 목록과 순서 있는 목록의 기본 구조 정리.
- 리스트·중첩 리스트 예제: 메뉴형 계층 구조 표현 방식 확인용.
- 엔티티: 공백, `<`, `>` 같은 특수 문자를 화면에 그대로 표시하는 방법 정리.
- Emmet 문법, 메뉴·레이아웃 구조: 짧은 축약 문법으로 반복 태그와 레이아웃 뼈대를 빠르게 만드는 흐름 정리.
- table, rowspan/colspan: 표 기본 구조와 셀 병합 방식 확인용.
- 테이블 기본 예제: 제목 셀과 일반 셀 구분 방식 확인용.
- 테이블 활용: 시간표나 게시판처럼 실제 데이터 배치 적용 방식 정리.
- a, img, target _blank / Emmet: 링크, 이미지, 새 창 열기, 빠른 마크업 생성 방식 정리.
- 레이아웃 구조 생성 예시: header, nav, main, footer 같은 큰 구역 선분리 습관 정리.
- video, MIME TYPE: 비디오 태그와 파일 형식 정보의 브라우저 재생 연결 구조 정리.
- 비디오 예제: controls, autoplay, muted, loop 같은 속성의 화면 표시 확인용.
- form, input, select, label: 입력 폼 핵심 태그의 역할과 연결 관계 정리.
- 폼 + CSS 기초: 입력창, 버튼, 라벨 기본 스타일링 연습용.
- form 기본 개념, input 종류: text, password, checkbox, radio 등 입력 타입별 용도 정리.
- 폼 + CSS 응용: 여백, 정렬, 색상에 따른 사용성 차이 확인용.
- form + Bootstrap: Bootstrap 컴포넌트 기반 폼 구성 흐름 정리.
- 폼 예제: 회원가입·로그인 같은 실제 서비스 입력 화면 형태 연습용.
- 폼 유효성 검증 (HTML5): required, pattern, disabled, readonly 같은 속성 활용 정리.
- data-* 속성 (DataSet): 사용자 정의 데이터를 저장하고 JavaScript에서 꺼내 쓰는 방법 정리.
- 기타 input type: 날짜, 시간, 숫자, 검색 같은 다양한 입력 타입 특징 확인용.
- 폼 연습: 입력 요소와 검증 방식을 묶어 하나의 완성형 폼으로 정리하는 단계.

## 01BASIC

- Block vs Inline (display): 요소 배치 방식 구분용.
- 스타일 우선순위: 여러 CSS 규칙이 겹칠 때 최종 적용 기준 확인용.
- Width / Height: 박스의 가로·세로 크기 지정 방식 정리.
- Color: 키워드, RGB, HEX 등 색상 표현 방법 정리.
- 단위: px, %, vw, vh 같은 크기 단위의 기준 비교용.
- Background: 배경색, 배경이미지, 반복 여부, 크기 조절 방식 정리.
- 텍스트 스타일: 정렬, 줄 높이, 글자 간격 등 문장 배치 방식 정리.
- Font Size: 고정 크기와 상대 크기 차이 정리.
- Font Family: 폰트 지정과 대체 서체 흐름 정리.

## 02BOX

- Margin: 요소 바깥쪽 여백과 가운데 정렬 원리 정리.
- Padding: 내용과 테두리 사이 간격 조절 방식 정리.
- Border: 선의 두께, 색상, 모양, radius를 이용한 박스 외형 구성 방식 정리.
- Box Sizing: width/height 계산 기준 비교용.
- Overflow: 내용이 박스를 넘칠 때 처리 방식 비교용.

## 03SELECTOR

- 선택자 기초: 태그, 클래스, 아이디로 원하는 요소를 고르는 기본 방법 정리.
- 자식/자손 선택자: 바로 아래 요소와 하위 전체 요소 구분용.
- 자식 선택자 레이아웃 예제: 부모 구조 변화에 따른 선택 결과 차이 확인용.
- 동위(형제) 선택자: 바로 옆 요소와 뒤쪽 형제 전체 선택 차이 정리.
- 속성 선택자: 속성값 기준 요소 선택 방식 정리.
- 의사 클래스, :checked + label: 입력 상태 변화에 따라 인접 라벨까지 함께 스타일링하는 흐름 정리.
- 의사 클래스 (:hover, :active, :visited): 마우스 올림, 클릭, 방문 상태별 화면 변화 정리.
- 의사 요소 (::before, ::after): 실제 태그 추가 없이 꾸밈 요소를 붙이는 방식 정리.
- :nth-child: 특정 순번 요소만 골라 스타일을 다르게 주는 방식 정리.

## 04POSITION

- 상대 위치 이동 예제: `relative` 를 사용해 원래 자리를 기준으로 `top`, `left` 값을 이동시키는 흐름 정리.
- 절대 위치 배치 예제: `absolute` 를 사용해 화면이나 position이 있는 부모를 기준으로 요소를 특정 좌표에 배치하는 흐름 정리.
- 가로 GNB 기본 구조: 메인 메뉴를 가로로 놓고 서브 메뉴를 아래에 숨겨 두는 기본 패턴 정리.
- 가로 GNB 1차 확장: hover 시 서브 메뉴가 나타나도록 부모·자식 위치 관계를 정리하는 단계.
- 가로 GNB 2차 확장: 메뉴 간격, 정렬, 서브 메뉴 시작 좌표를 다듬는 흐름 정리.
- 세로 GNB 기본 구조: 메뉴를 세로 방향으로 쌓고 서브 메뉴를 오른쪽으로 펼치는 형태 정리.
- 세로 GNB 응용: 가로 메뉴와 세로 메뉴의 `absolute` 배치 차이 비교용.
- GNB 애니메이션 응용: transition을 더해 메뉴가 자연스럽게 열리는 방식 정리.
- fixed 예제: 스크롤과 무관하게 같은 자리에 남는 고정 메뉴나 버튼 구성 방식 정리.
- sticky 예제: 처음에는 일반 흐름에 있다가 특정 지점부터 고정되는 동작 확인용.
- position 종합 비교: static, relative, absolute, fixed, sticky의 기준점 차이 정리.
- GNB 연습 초안: position 규칙으로 메뉴 구조를 직접 조립해 보는 단계.
- GNB 연습 응용: 메뉴 위치, 서브 메뉴 방향, hover 처리를 묶어 완성도 있게 다듬는 단계.

## 05LAYOUT

- Flexbox 기초: 주축과 교차축 개념, 정렬 방식, 간격 조절 정리.
- Flexbox 심화: grow, shrink, basis, align-self 같은 자식 단위 제어 확장 정리.
- Flex 연습: 여러 박스를 원하는 방향과 간격으로 배치하는 문제 풀이용.
- Flex 연습 정답: 같은 배치를 더 깔끔한 속성 조합으로 만드는 방법 비교용.
- Grid: 행과 열을 동시에 다루는 2차원 레이아웃 방식 정리.
- KREAM 스타일 레이아웃: 실무형 화면 구조를 레이아웃에 적용하는 연습용.

## 06ANIMATION

- Transition 기초: hover 같은 상태 변화가 즉시 바뀌지 않고 부드럽게 이어지도록 만드는 방식 정리.
- Transition 이동 응용: 크기나 색상뿐 아니라 위치 변화도 시간차를 두고 움직이게 만드는 흐름 정리.
- Transform 기초: 이동, 확대, 회전, 기울이기 같은 변형 정리.
- Transition + Transform 응용: 움직임과 시간 변화 결합 시 화면 체감 변화 비교용.
- Transform 3D 기초: perspective 와 rotateX/rotateY 를 통한 입체 반전 효과 정리.
- Transform 3D 응용: 3D 효과를 메뉴나 카드처럼 실제 UI에 적용하는 방식 정리.
- Animation (@keyframes) 기초: 중간 단계까지 지정해 반복 움직임을 만드는 방식 정리.
- Animation 응용 1: duration, direction, iteration-count 차이에 따른 체감 변화 비교용.
- Animation 응용 2: 색상과 위치가 함께 변하는 다단계 움직임 정리.
- Animation 응용 3: 공 튀기기나 배너 이동 같은 실제 화면 효과 연결용.
- Animation 응용 4: hover 시 멈춤, 강조 확대처럼 사용자 반응까지 묶는 흐름 정리.


## 07MQ

- 미디어 쿼리: 화면 너비에 따라 다른 CSS를 적용해 반응형 화면을 만드는 방식 정리.
- 미디어 쿼리 분리형: 한 파일에 모두 쓰는 방식과 CSS 파일을 나누는 방식 비교용.


### JavaScript 핵심 요약
- 1. **실행 위치** : `onclick`, `href="javascript:..."`, `<script>` 내부
- 2. **DOM** : `document.getElementById`, `querySelector`, `innerHTML`, `document.write`
- 3. **보간법** : 백틱(``) + `${표현식}`
- 4. **변수** : `let`(재할당 가능), `const`(상수)
- 5. **타입** : number, string, boolean, object, null, undefined (typeof로 확인)
- 6. **객체** : `{ key: value }`, 메서드, `this`
- 7. **배열** : `[]`, index 접근, `push`/`pop`, `forEach`, `filter`, `map`, `reduce`
- 8. **입력값 처리** : `querySelector`, `.value`, `Number()`
- 9. **동적 DOM 생성** : `createElement`, `appendChild`, `Object.entries`
- 10. **학습 범위** : JAVASCRIPT 폴더의 기초 문법, 타입, DOM, 동적 생성 흐름까지의 범위 정리.
