# 5월 21일 필기 — JSP

## 핵심요약

Ch01은 JSP의 세 가지 기본 문법 요소를 학습했다. **선언문**(`<%! %>`)은 JSP가 Servlet으로 변환될 때 클래스 레벨에 추가되어 인스턴스 변수·메서드로 동작하므로 요청 간 상태가 유지된다. **스크립틀릿**(`<% %>`)은 `service()` 함수 내부에서 처리되어 지역변수로 동작하고 요청마다 초기화된다. **표현식**(`<%= %>`)은 서블릿 값을 HTML로 출력할 때 사용한다. C04·C05에서는 스크립틀릿 블록을 여닫으면서 HTML 태그와 섞어 중첩 반복문으로 동적 테이블을 생성하는 패턴을 연습했다. Ch02는 `request` 내장객체로 폼 데이터를 수신하는 방법을 단계적으로 다뤘다. C01에서 GET 방식으로 `request.getParameter()`를 이용해 단일 파라미터를 받고, C02~C03에서 POST 방식과 한글 인코딩(`request.setCharacterEncoding("UTF-8")`) 처리를 추가했다. C04~C05에서는 `<jsp:useBean>`과 `<jsp:setProperty property="*">`를 조합해 폼 파라미터를 DTO에 자동 바인딩하는 방법을 학습했다. 이때 DTO 필드명과 HTML `name` 속성값이 일치해야 자동 매핑이 된다. C06에서는 JSP 스크립틀릿 변수를 JavaScript 변수에 담는 방식과 `request.setAttribute()`로 저장한 값을 EL(`${ }`)로 가져오는 두 가지 방식을 비교했다.

### 2) C01 — 선언문 기본

```jsp
<%!
    String name = "HONG GIL DONG";
    public String testFunc(){
        System.out.println("선언문 함수 테스트!");

```

### 4) C03 — 스크립틀릿 기본

```jsp
<%
    String str1 = "HELL01";
```

### 5) C04 — 스크립틀릿 + HTML 혼합 (2중 반복 테이블)

```jsp
<%@ page import="java.util.*" %>

    Scanner sc = new Scanner(System.in);

    sc.close();
%>

        <tr>
```

## JSP01 — Ch02 Request 내장객체

---

### 1) 핵심 개념

| 개념 | 사용 위치 | 포인트 |
|---|---|---|
| **GET 파라미터** | `C01` | URL Query String → `request.getParameter()` |
| **POST 파라미터** | `C02` | Request Body → 동일하게 `request.getParameter()` |
| **한글 인코딩** | `C03` | `request.setCharacterEncoding("UTF-8")` — 파라미터 조회 전 호출 |
| **DTO 자동 바인딩** | `C04~C05` | `<jsp:useBean>` + `<jsp:setProperty property="*">` |
| **JSP → JavaScript** | `C06` | 표현식으로 직접 주입 / `setAttribute` + EL |

---

### 2) C01 — GET 방식 파라미터 수신

```html
<!-- C01Request_Get.html -->
<form action="./C01Request_Process.jsp" method="get">
    <input type="text"     name="username" />
    <input type="password" name="password" />
    <input type="text"     name="bgcolor"  />
    <input type="submit" />
</form>
```

```jsp
<!-- C01Request_Process.jsp -->
<%
    /*
        request 내장객체 : 요청정보값 저장 (HttpServletRequest 클래스로 생성된 객체)
    */
    String username = request.getParameter("username");
    String password = request.getParameter("password");
    String bgColor  = request.getParameter("bgcolor");
%>
<body bgcolor=<%=bgColor %>>
    username : <%=username %><br>
    password : <%=password %><br>
    bgcolor  : <%=bgColor %><br>
</body>
```

### 4) C03 — POST + 한글 인코딩 처리

```jsp
<%
    request.setCharacterEncoding("UTF-8");
    response.setContentType("text/html; charset=UTF-8");

    String username = request.getParameter("username");
    String password = request.getParameter("password");
    String bgColor  = request.getParameter("bgcolor");
%>
```

> `request.setCharacterEncoding("UTF-8")` 은 **파라미터 조회 전**에 반드시 호출해야 함

### 5) C04 — useBean / DTO 자동 바인딩

```java
// C04TestDto.java
package Ch02;

public class C04TestDto {
    private String username;
    private String password;
    private String bgcolor;
    // 디폴트 생성자, 모든인자 생성자, Getter/Setter, toString
}
```

```jsp
<!-- C04Request_Process.jsp -->
<%
    request.setCharacterEncoding("UTF-8");
    response.setContentType("text/html; charset=UTF-8");
%>
<jsp:useBean id="dto" class="Ch02.C04TestDto" scope="request" />
<jsp:setProperty property="*" name="dto"/>
<%
    System.out.println("DTO : " + dto);
%>
<body bgcolor=<%=dto.getBgcolor() %>>
    username : <%=dto.getUsername() %><br>
    password : <%=dto.getPassword() %><br>
    bgcolor  : <%=dto.getBgcolor() %><br>
</body>
```

| 태그 | 설명 |
|---|---|
| `<jsp:useBean id="dto" class="패키지.클래스" scope="request">` | Bean 객체 생성/재사용 |
| `<jsp:setProperty property="*" name="dto">` | 폼 파라미터명과 동일한 필드에 자동 매핑 |

> DTO 조건: 디폴트 생성자 + Getter/Setter 필수, **필드명 = HTML `name` 속성값** 일치 시 자동 매핑

### 7) C06 — JSP → JavaScript 데이터 전달

```jsp
<%
    String msg1 = "HELLO WORLD1";
    String msg2 = "HELLO WORLD2";

    request.setAttribute("msg3", "HELLO WORLD3");
    request.setAttribute("msg4", "HELLO WORLD4");
%>

<script>
    const m1 = '<%=msg1%>';   // 스크립틀릿 표현식
    const m2 = '<%=msg2%>';   // 스크립틀릿 표현식

    const m3 = '${msg3}';     // EL (request.setAttribute 값)
    const m4 = '${msg4}';     // EL

    console.log('m1', m1);
    console.log('m2', m2);
    console.log('m3', m3);
    console.log('m4', m4);
</script>
```

| 방식 | 문법 | 데이터 출처 |
|---|---|---|
| **표현식** | `'<%=변수%>'` | 스크립틀릿 지역변수 |
| **EL** | `'${키}'` | `request.setAttribute()` 로 저장한 값 |
