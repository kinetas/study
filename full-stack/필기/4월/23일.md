# 4월 23일 필기 — JAVA

## 핵심요약

이중 while은 외부 루프가 행(row), 내부 루프가 열(col)을 제어한다. 구구단은 단 변수와 배수 변수를 각각 외·내부 루프로 관리하며, 입력 검증도 while로 재입력을 반복한다. 별찍기의 핵심은 출력 개수를 `i`에 대한 수식으로 표현하는 것이며, 공백 포함 패턴은 공백 루프 → 별 루프 순으로 구성한다. 다이아몬드는 꺾임 카운터 `l` 방식(고정 높이)과 `spaces/stars/dir` 방향 변수 방식(변수 높이) 두 가지로 구현한다.

## JAVA — 이중 while / 구구단 / 별찍기

---

### 1) 이중 while 구조

```
while (외부 조건) {       ← 행(row) 제어
    while (내부 조건) {   ← 열(col) 제어
        // 한 칸 출력
        j++;
    }
    System.out.println(); ← 줄바꿈
    i++;
}
```

| 루프 | 역할 | 변수 |
|------|------|------|
| 외부 | 몇 줄 반복할지 | `i` |
| 내부 | 한 줄에 몇 개 출력할지 | `j` |

### 2) CH06 — C02 구구단

```java
// 2~9단 전체
int dan = 2;
while (dan < 10) {
    int i = 1;
    while (i < 10) {
        System.out.printf("%d x %d = %d\t", dan, i, dan * i);
        i++;
    }
    dan++;
    System.out.println();
}

// n~9단 (입력 검증 포함)
Scanner sc = new Scanner(System.in);
int n = sc.nextInt();
while (n > 9 || n < 0) {
    System.out.print("2이상 9이하 값 입력 : ");
    n = sc.nextInt();
}
while (n < 10) {
    int i = 1;
    while (i < 10) {
        System.out.printf("%d x %d = %d\t", n, i, n * i);
        i++;
    }
    n++;
    System.out.println();
}
sc.close();

// n~m단 (n < m, 2 이상 9 이하 검증)
int n = sc.nextInt(), m = sc.nextInt();
while (n > 9 || n < 2 || n >= m || m <= 2 || m >= 10) {
    System.out.print("2이상 9이하 값 입력 or n>m 불만족 : ");
    n = sc.nextInt();
    m = sc.nextInt();
}
while (n <= m) {
    int i = 1;
    while (i < 10) {
        System.out.printf("%d x %d = %d\t", n, i, n * i);
        i++;
    }
    n++;
    System.out.println();
}

// n~m단 역순 출력 (각 단 내부 9→1)
while (n <= m) {
    int i = 9;
    while (i > 0) {
        System.out.printf("%d x %d = %d\t", n, i, n * i);
        i--;
    }
    n++;
    System.out.println();
}

// n~m단 역순 출력 (m단부터 n단 방향, 각 단 내부 9→1)
while (n < m) {
    int i = 9;
    while (i > 0) {
        System.out.printf("%d x %d = %d\t", m, i, m * i);
        i--;
    }
    m--;
    System.out.println();
}
```

> **입력 검증 패턴** : 조건이 틀릴 때 재입력 받는 while — 단순 if 대신 while을 써야 반복 재입력이 가능

### 4) CH06 — C02 별찍기 (공백 포함)

```
11) 정삼각형          12) 역삼각형
   *                  *******
  ***                  *****
 *****                  ***
*******                  *
공백: h-i-1             공백: i
별:   2*i+1             별:   2*h-2*i-1
```
```java
// 11) 정삼각형
int h = 4, i = 0;
while (i < h) {
    int j = 0;
    while (j < h - i - 1) { System.out.print(" "); j++; }
    int k = 0;
    while (k < 2 * i + 1) { System.out.print("*"); k++; }
    System.out.println();
    i++;
}

// 12) 역삼각형
int h = 4, i = 0;
while (i < h) {
    int j = 0;
    while (j < i) { System.out.print(" "); j++; }
    int k = 0;
    while (k < 2 * h - 2 * i - 1) { System.out.print("*"); k++; }
    System.out.println();
    i++;
}
```

```
13) 다이아몬드 고정 (공백 포함, 높이 7)
   *                공백: 3-i+2*l
  ***               별:   2*i-4*l+1
 *****              i>=4 부터 l++ (매 행)
*******
 *****
  ***
   *
```
```java
int i = 0, l = 0;
while (i < 7) {
    int j = 0;
    while (j < 3 - i + 2 * l) { System.out.print(" "); j++; }
    int k = 0;
    while (k < 2 * i - 4 * l + 1) { System.out.print("*"); k++; }
    System.out.println();
    i++;
    if (i >= 4) l++;
}
```

---

```
14) 다이아몬드 변수 h (공백 포함)
초기: spaces=(h-1)/2, stars=1, dir=1
```
```java
int h = 8, i = 0;
int spaces = (h - 1) / 2, stars = 1, dir = 1;
while (i < h) {
    int j = 0;
    while (j < spaces) { System.out.print(" "); j++; }
    int k = 0;
    while (k < stars) { System.out.print("*"); k++; }
    System.out.println();
    i++;
    if (dir == 1) {
        if (i > h / 2)       { dir = -1; spaces++; stars -= 2; }
        else if (spaces > 0) { spaces--; stars += 2; }
    } else { spaces++; stars -= 2; }
}
```

---

```
15) 역다이아몬드 변수 h (공백 포함)
*******     초기: spaces=0, stars=h, dir=1
 *****
  ***
   *
  ***
 *****
*******
```
```java
int h = 7, i = 0;
int spaces = 0, stars = h, dir = 1;
while (i < h) {
    int j = 0;
    while (j < spaces) { System.out.print(" "); j++; }
    int k = 0;
    while (k < stars) { System.out.print("*"); k++; }
    System.out.println();
    i++;
    if (dir == 1) {
        if (i > h / 2)                  { dir = -1; spaces--; stars += 2; }
        else if (spaces < (h - 1) / 2) { spaces++; stars -= 2; }
    } else { spaces--; stars += 2; }
}
```

> **별찍기 수식 요약**
>
> | 패턴 | 공백 | 별 |
> |------|------|----|
> | 좌측 계단 | — | `i+1` |
> | 역계단 | — | `h-i` |
> | 홀수 증가 | — | `2*i+1` |
> | 홀수 감소 | — | `(2*h)-(2*i)-1` |
> | 정삼각형 | `h-i-1` | `2*i+1` |
> | 역삼각형 | `i` | `2*h-2*i-1` |
> | 다이아몬드 고정 | `3-i+2*l` | `2*i-4*l+1` |
> | 다이아몬드 변수 | `spaces/stars/dir` 방식 | 방향 전환으로 조정 |
