### 4) CH00 — HelloWorld

```java
package CH00;   // 패키지명

public class HelloWorld   // 클래스 선언부 - 객체지향문법 적용영역
{
    // 메서드(함수) 종류
    // 라이브러리 메서드  : 미리 만들어져 제공되는 메서드
    // 사용자정의 메서드  : 개발자에 의해 만들어지는 메서드
    // main 메서드       : 최초 실행되는 메서드

    public static void main(String[] args) {   // main 메서드 선언부 - 절차지향문법 적용영역
        System.out.println("Hello, Java!");    // 라이브러리 메서드
    }
    // 글자크기조정 : ctrl + '+' or '-'
    // 주석처리   : ctrl + '/'
    // 한줄복사   : ctrl + alt + '↓'
    // 한줄삭제   : ctrl + d
}
```

### 5) CH01 — C01SystemOut

출력 메서드 및 서식문자

| 서식문자 | 설명 |
|----------|------|
| `%d` | 10진수 정수 |
| `%f` | 10진수 실수 |
| `%c` | 한 문자 |
| `%s` | 문자열 |

이스케이프 문자

| 문자 | 설명 |
|------|------|
| `\n` | 개행, 줄바꿈 |
| `\b` | 백스페이스 |
| `\t` | 탭길이(기본 8칸)만큼 커서 이동 |

```java
System.out.printf("%d %d %d \n", 10, 20, 30);
System.out.printf("%f %f %f \n", 10.1, 20.1, 30.1);
System.out.printf("%c %c %c \n", 'A', 'B', 'C');
System.out.printf("%s %s %s \n", "This is", " String ", "Test");
System.out.printf("%d.%s : %d\n", 1, "국어", 100);
```

### 6) CH01 — C02진수

진수 접두사

| 진수 | 접두사 | 서식문자 | 예시 |
|------|--------|----------|------|
| 2진수 | `0b` | `%d` | `0b00001111` |
| 8진수 | `0` | `%o` | `0255` |
| 10진수 | 없음 | `%d` | `173` |
| 16진수 | `0x` | `%x` | `0xAD` |

비트 크기표

| bit | 경우의 수 | 범위 |
|-----|-----------|------|
| 1bit | 2¹ = 2 | 0 ~ 1 |
| 4bit | 2⁴ = 16 | 0 ~ 15 |
| 8bit | 2⁸ = 256 | 0 ~ 255 |

> 코드 자동정렬 : `ctrl + shift + f`

```java
System.out.printf("10진수 : %d\n", 0b00001111); // 2진수
System.out.printf("10진수 : %d\n", 173);         // 10진수
System.out.printf("10진수 : %d\n", 0255);        // 8진수
System.out.printf("10진수 : %d\n", 0xAD);        // 16진수
```

### 7) CH01 — C03음수

> 컴퓨터(CPU)는 덧셈연산만 가능 → **보수 개념 + 가산처리**로 뺄셈 결과를 만들어 낸다

**보수 계산 원리**

```
00000101  =  5
11111010  =  5에 대한 1의 보수 (-6)
11111011  =  5에 대한 2의 보수 (-5)   ← 1의 보수 + 1
```

**연습 문제**

| 10진수 | 2진수 |
|--------|-------|
| 111 | 01101111 |
| -111 | 10010001 |
| 96 | 01100000 |
| -96 | 10100000 |
| 31 | 00011111 |
| -31 | 11100001 |

| 2진수 | 10진수 |
|-------|--------|
| 10101111 | -81 |
| 00110101 | 53 |
| 11001100 | -52 |
| 10101010 | -86 |
