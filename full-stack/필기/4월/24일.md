# 4월 24일 필기 — JAVA

## 핵심요약

`continue`는 현재 반복만 건너뛰고, `break`는 반복문을 즉시 탈출한다. 중첩 반복문 전체 탈출은 flag 변수 방식과 레이블(`Exit:`) 방식 두 가지로 구현할 수 있다. `for`문은 초기화·조건·증감을 한 줄에 표현하며, 슬롯을 비워두면 입력 검증에도 활용할 수 있다. 컬렉션 순회는 기본 for / 개량 for / 스트림 람다 세 가지 방식이 있다. 객체지향에서 클래스는 속성(멤버 변수)과 기능(메서드)으로 구성되며, `new`로 힙에 객체를 생성하고 참조변수가 스택에서 주소를 가리킨다. 변수는 멤버·지역·파라미터·static 네 종류로 구분하며, 이름이 겹칠 때 `this.`로 멤버 변수를 명시한다.

### 1) break

```java
// -1 입력 전까지 누적합
Scanner sc = new Scanner(System.in);
int n = 0, sum = 0;
while (true) {
    n = sc.nextInt();
    if (n == -1) break;
    sum += n;
}
System.out.println("SUM : " + sum);
sc.close();

// 한 줄 축약
while ((n = sc.nextInt()) != -1) sum += n;

// flag로 이중 while 전체 탈출 (7×7까지)
int dan = 2;
boolean isExit = false;
while (dan < 10) {
    int i = 1;
    while (i < 10) {
        System.out.printf("%d x %d = %d\t", dan, i, dan * i);
        if (dan == 7 && i == 7) { isExit = true; break; }
        i++;
    }
    if (isExit) break;
    dan++;
    System.out.println();
}

// 레이블(label)로 이중 while 전체 탈출
int dan = 2;
Exit:
while (dan < 10) {
    int i = 1;
    while (i < 10) {
        System.out.printf("%d x %d = %d\t", dan, i, dan * i);
        if (dan == 7 && i == 7) break Exit;
        i++;
    }
    dan++;
    System.out.println();
}
```

> **flag vs 레이블** : flag는 변수 선언이 필요하고 if로 재확인해야 하지만, `레이블:` + `break 레이블`은 한 줄로 전체 탈출 가능

### 2) continue

```java
// 1~10 중 3의 배수 제외하고 합 출력
int i = 1, sum = 0;
while (i <= 10) {
    if (i % 3 == 0) { i++; continue; }
    System.out.println("i : " + i);
    sum += i;
    i++;
}
System.out.println("SUM : " + sum);
```

> **break vs continue**
>
> | 키워드 | 동작 | 사용 예 |
> |--------|------|---------|
> | `break` | 반복문 즉시 탈출 | 특정 조건 충족 시 종료 |
> | `continue` | 이후 코드 건너뛰고 다음 반복 | 특정 값 스킵 |
> | `break 레이블` | 중첩 반복문 전체 탈출 | 이중 for/while 동시 탈출 |

### 3) CH06 — C04 for문 기본

```
for (초기화; 조건식; 증감식) { }
```

| | while | for |
|---|-------|-----|
| 구조 | 초기화·증감식이 외부에 분리 | 3요소를 한 줄에 표현 |
| 적합 | update가 복잡한 경우 | 고정 횟수 반복 |
| 빈 슬롯 | — | `for (; 조건; 업데이트)` 가능 |

```java
// 빈 초기화·증감 — 입력 검증
Scanner sc = new Scanner(System.in);
int n = sc.nextInt();
for (; n > 9 || n < 2; n = sc.nextInt()) {
    System.out.print("2이상 9이하 값 입력 : ");
}

// 홀수만 받기
int h = sc.nextInt();
for (; h % 2 == 0 || h <= 1; h = sc.nextInt()) {
    System.out.println("재입력(홀수) : ");
}
```

```java
    n = sc.nextInt(); m = sc.nextInt();

    }
    System.out.println();
}

// n~m단 역순 (각 단 9→1)
for (; n <= m; n++) {
    for (int i = 9; i > 0; i--) {
        System.out.printf("%d x %d = %d\t", n, i, n * i);
    }
    System.out.println();
}

// n~m단 역순 (m단→n단 방향)
for (; n < m; m--) {
    for (int i = 9; i > 0; i--) {
        System.out.printf("%d x %d = %d\t", m, i, m * i);
    }
    System.out.println();
}
```

```java
int l = 0;
for (int i = 0; i < 7; i++) {

// 13) 다이아몬드 고정 — if-else 분기 방식
for (int i = 0; i < 7; i++) {
    if (i < 4) {
        for (int j = 0; j < 3 - i; j++) System.out.print(" ");
        for (int k = 0; k <= 2 * i; k++) System.out.print("*");
    } else {
        for (int j = 0; j < i - 3; j++) System.out.print(" ");
        for (int k = 0; k <= 2 * (7 - i - 1); k++) System.out.print("*");
    }
    System.out.println();
}

// 14) 다이아몬드 변수 h — spaces/stars/dir 방식
int h = 8;
int spaces = (h - 1) / 2, stars = 1, dir = 1;
for (int i = 0; i < h; i++) {
    for (int j = 0; j < spaces; j++) System.out.print(" ");
    for (int k = 0; k < stars; k++) System.out.print("*");
    System.out.println();
    if (dir == 1) {
        if (i >= h / 2)      { dir = -1; spaces++; stars -= 2; }
        else if (spaces > 0) { spaces--; stars += 2; }
    } else { spaces++; stars -= 2; }
}

// 14) 다이아몬드 변수 h — if-else 분기 방식 (홀수 h만)
for (int i = 0; i < h; i++) {
    if (i < h / 2 + 1) {
        for (int j = 0; j < h / 2 - i; j++) System.out.print(" ");
        for (int k = 0; k <= 2 * i; k++) System.out.print("*");
    } else {
        for (int j = 0; j < i - h / 2; j++) System.out.print(" ");
        for (int k = 0; k <= 2 * (h - i - 1); k++) System.out.print("*");
    }
    System.out.println();
}

// 15) 역다이아몬드 변수 h — spaces/stars/dir 방식
int h = 7;
int spaces = 0, stars = h, dir = 1;
for (int i = 0; i < h; i++) {
    for (int j = 0; j < spaces; j++) System.out.print(" ");
    for (int k = 0; k < stars; k++) System.out.print("*");
    System.out.println();
    if (dir == 1) {
        if (i >= h / 2)                  { dir = -1; spaces--; stars += 2; }
        else if (spaces < (h - 1) / 2) { spaces++; stars -= 2; }
    } else { spaces--; stars += 2; }
}

// 15) 역다이아몬드 변수 h — if-else 분기 방식
int h = 7;
for (int i = 0; i < h; i++) {
    if (i < h / 2 + 1) {
        for (int j = 0; j < i; j++) System.out.print(" ");
        for (int k = 0; k < h - 2 * i; k++) System.out.print("*");
    } else {
        for (int j = 0; j < h - i - 1; j++) System.out.print(" ");
        for (int k = 0; k < 2 * (i + 1) - h; k++) System.out.print("*");
    }
    System.out.println();
}
```

> **while → for 변환 포인트**  
> - `i++` 위치가 for 증감식으로 이동 → `continue` 사용 시 `i++` 누락 없음

### 1) OOP 개념 / 클래스 / 메모리 구조

#### 객체(Object)

> 독점적이고 배타적인 공간(VOLUME)을 차지하는 사물

| 구성 요소 | 설명 | Java 표현 |
|-----------|------|-----------|
| **속성 (Attribute)** | 객체마다 구별되는 데이터 | 멤버 변수 (필드) |
| **기능 (Function)** | 객체가 수행할 수 있는 공통 ACTION | 메서드 (함수) |

#### 객체지향(OOP)

현실 세계의 객체 정보 중 문제 해결에 필요한 일부만 **추출(추상화)** 하여 메모리에 적재하는 문법 체계

#### 클래스(Class)

동일한 종류의 객체에 필요한 메모리 공간을 제공하기 위해 선언된 **자료형**  
→ 객체 생성 전에는 메모리를 차지하지 않음, `new`로 생성하는 순간 속성·기능대로 공간 형성

#### 자바의 메모리 영역

| 영역 | 설명 | 저장 대상 |
|------|------|-----------|
| **스택(Stack)** | `{}` 내에서 생성·소멸 | 지역 변수, 기본 타입 변수 |
| **클래스(=메서드) 영역** | 공유 메모리 | 생성자, 일반·static 메서드, static 변수 |
| **힙(Heap)** | 객체 저장 영역 | `new` 예약어 사용 시 할당 |

```java
Scanner sc = new Scanner(System.in);
//  ↑참조변수(Stack)  ↑Heap에 객체 생성  ↑생성자(초기값 부여)
// new → Heap에 Scanner 객체 생성 → 시작 주소를 Stack의 sc에 저장
```

### 3) CH01 — C03 메서드 추가 (기능)

```java
class Person {
    public String name;
    public int age;
    public float height;
    public double weight;

    void talk() { System.out.printf("%s님이 말합니다.\n", this.name); }
    void walk() { System.out.printf("%s님이 걷습니다.\n", this.name); }

    String showInfo() {
        return "name : " + this.name + " age : " + this.age +
               " height : " + this.height + " weight : " + this.weight;
    }

    @Override
    public String toString() {
        return "Person [name=" + name + ", age=" + age +
               ", height=" + height + ", weight=" + weight + "]";
    }
}

Person hong = new Person();
hong.name = "홍길동"; hong.age = 15; hong.height = 177.5f; hong.weight = 70.5;

hong.talk();                          // 홍길동님이 말합니다.
hong.walk();                          // 홍길동님이 걷습니다.
System.out.println(hong.showInfo());  // name : 홍길동 age : 15 ...
System.out.println(hong.toString());  // Person [name=홍길동, age=15, ...]
```

> **`@Override toString()`** : `System.out.println(객체)` 시 자동 호출 — 객체 상태를 문자열로 확인할 때 오버라이드

### 5) CH01 — C05 변수 종류

| 변수 종류 | 선언 위치 | 생존 범위 | 용도 |
|-----------|-----------|-----------|------|
| **멤버 변수** | 클래스 내부, 메서드 외부 | 객체 생존 동안 | 객체 개별 데이터 저장 |
| **지역 변수** | 메서드(또는 `{}`) 내부 | 해당 `{}` 종료 시 소멸 | 임시 계산·처리 |
| **파라미터 변수** | 메서드 선언부 `()` 안 | 해당 메서드 내부 | 외부에서 값 전달 수신 |
| **static 변수** | 클래스 내부 (`static` 키워드) | 프로그램 종료 시까지 | 동일 클래스 객체 간 공유 |

```java
class Simple {
    int num = 10; // 멤버 변수

    void Func1() {
        num++;                     // 멤버 변수 → 11
        System.out.println(num);   // 11
        int num = 20;              // 지역 변수 선언 (멤버 변수와 이름 중복)
        num++;                     // 지역 변수 → 21
        this.num++;                // 멤버 변수 → 12
        System.out.println(num);       // 21 (지역 변수)
        System.out.println(this.num);  // 12 (멤버 변수)
    }

    void Func2() {
        System.out.println(num);   // 멤버 변수
        if (num > 0) {
            int num = 100;             // if 블록 지역 변수
            System.out.println(num);   // 100
        }
        System.out.println(num);   // 다시 멤버 변수
        while (num < 15) {
            int num = 100;             // while 블록 지역 변수
            System.out.println(num);   // 100
            num++;                     // 지역 변수 ++
            this.num++;                // 멤버 변수 ++
        }
    }

    void Func3(int num) {  // 파라미터 변수
        this.num = num;    // 파라미터 → 멤버 변수에 대입
    }
}
```

> **`this.num` vs `num`** : 같은 이름이 겹칠 때 `this.`를 붙이면 멤버 변수, 없으면 가장 가까운 지역 변수
